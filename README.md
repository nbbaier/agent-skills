# Agent Skills

A collection of custom skills I've put together for coding agents.

## Skills

| Skill                                           | Description                                                                                               |
| ----------------------------------------------- | --------------------------------------------------------------------------------------------------------- |
| [ideation](skills/ideation/)                           | Guides structured ideation through Socratic questioning to explore problems, opportunities, and solutions |
| [implementation-guide](skills/implementation-guide/)   | Generates comprehensive step-by-step implementation guides instead of writing code directly               |
| [react-effect-patterns](skills/react-effect-patterns/) | Guidelines for proper React useEffect usage and avoiding unnecessary Effects                              |
| [val-town-cli](skills/val-town-cli/)                   | Reference for managing Val Town projects using the `vt` CLI                                               |

---

## Installing Skills

Copy any skill directory to your agent's skills location or using Vercel's `skills` CLI:

```bash
npx skills add nbbaier/agent-skills
```
